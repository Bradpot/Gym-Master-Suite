**Gym Master Suite**
A modern, responsive, and feature-rich Gym Management System built to simplify gym operations. Gym Master Suite provides an intuitive interface for managing members, trainers, memberships, attendance, payments, workout plans, and administrative tasks in one centralized platform.

**Overview**

Gym Master Suite is designed to help gym owners and administrators efficiently manage daily operations while providing a seamless experience for members and trainers. The application focuses on automation, user-friendly design, and organized data management.

**Installation**
Clone the repository
-git clone https://github.com/yourusername/Gym-Master-Suite.git


Navigate into the project
-cd Gym-Master-Suite


Create Virtual Environment (Optional)
-python -m venv venv


Activate virtual environment
Windows
-venv\Scripts\activate

Mac/Linux
-source venv/bin/activate


**Install Dependencies**
-pip install -r requirements.txt


**Run the Application**
-python app.py


Open your browser and visit
-http://127.0.0.1:5000/
